# ProductProvenance

| --------- | ----------------------- | -------- | ------------------------------------------- |
| creator   | [Creator](creator.md)   | ✅        | Media url of the product                    |

| token     | [Token](token.md)       | ❌        | Link to the token related to the product    |
| networkId | number                  | ❌        | Network ID of the product (if applicable)   |
